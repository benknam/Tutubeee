# Tutubeee 图片整理助手

<img width="2890" height="3303" alt="image" src="https://github.com/user-attachments/assets/82472b3d-8d61-46e1-a720-9b0df8776670" />


一款可以快速让原本杂乱无章的图片进行分类整理的工具，使用户可以方便对安卓手机中的照片或图片进行整理归档以及备份。

功能特点：

1. 按文件时间整理：

   设定了按年、按月、按日三个方式，将照片按照不同的时间放入分类文件夹中；

   根据不同的照片或图片逻辑设定不同的整理方式。
   

3. 按文件名整理：

   可使用关键字词对目标文件夹进行搜索并进行后续文件操作；


4. 根据EXIF信息进行整理：

   可根据照片或图片的EXIF数据进行搜索和整理，目前设定了几个比较常见的参数作为条件，分别为

   设备品牌、设备型号、光圈、焦距、曝光程序


5. 兼容性和技巧：
   
   目前已完成对安卓5～安卓13的适配，意味着我们旧手机里的照片都可以拿来归档备份了。

   对于支持OTG的安卓手机来说，用这个程序可以快速整理外部SD卡的照片图片，简直是神器。
