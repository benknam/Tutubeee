# Tutubeee 图片整理助手

是一款可以快速让原本杂乱无章的图片进行分类整理的工具，使用户可以方便对手机中的照片或图片进行归档备份。

功能特点：

1. 按文件时间整理：

   设定了按年、按月、按日三个方式，将照片按照不同的时间放入分类文件夹中；

3. 按文件名整理：

   可使用关键字词对目标文件夹进行搜索并进行后续文件操作；

4. 兼容性：
   
   目前已完成对安卓5～安卓13的适配，意味着我们旧手机里的照片都可以拿来归档备份了。

   Windows/Mac客户端正在同步开发中，即将加入针对EXIF信息读取和整理归档的功能。
